# Install dependencies

pnpm i

Add OPENAI_API_KEY and DEEPGRAM_API_KEY keys in .env

pnpm dev-server
pnpm dev-pods

# Use frontend with browser microphone:

https://github.com/restackio/voice-frontend

pnpm dev

# Or use Twilio voice calls

## Start ngrok funnel

pnpm ngrok

## Trigger Twilio call with

pnpm call
